How we made this site



Web Programming

 When dealing with websites and programming for the web, there are
 some significant differences from standard console development.
 For one, websites typically require multiple languages: HTML (Hypertext
 Markup Language) for the page structure, CSS (Cascading Style Sheets)
 for visual styling, and JavaScript for interactivity. Rather than being
 compiled ahead of time, these are interpreted at runtime by your web
 browser to produce a website similar to what you see now. In terms of
 development, we used Visual Studio Code as our primary editor and
 collaborated using GitHub.
