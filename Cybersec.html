Cyber Security

Cybersecurity as a focus

 Cyber Security focuses on establishing, maintaining, and securing
 networks for a professional environment. You will learn different
 tools used in industry, as well as your expected programming classes.
 Cyber Security takes a different approach to programming, since you
 will not be building games but instead learning how to build your own
 tools that may be useful to you in the future.
